## Steps:

### Start the project

```sh
pnpm create @quick-start/electron
```

### install

```sh
pnpm i
```

### Add vue devtools

```sh
# Add vuedevtools
pnpm install --save-dev @vue/devtools
```

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

$ pnpm build:linux
```
